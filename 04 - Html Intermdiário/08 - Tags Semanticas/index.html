Section

A tag <section> é usada para agrupar conteúdo relacionado dentro de uma página web. Ela representa uma seção temática do documento.

Exemplo de uso da tag <section>:

Seção 1: Introdução

Esta é a introdução do meu artigo.

Seção 2: Desenvolvimento

Aqui está o desenvolvimento do meu artigo.

Seção 3: Conclusão

Esta é a conclusão do meu artigo.

Footer

A tag <footer> é usada para definir o rodapé de um documento ou seção.
